# HiveInsight

HiveInsight is een webapplicatie die gebruikers in staat stelt om hun slimme bijenkorf te verbinden en te beheren.

## Vereisten

- Webserver (zoals Apache)
- PHP
- mysqli PHP extensie
- MySQL Server


## Installatie

1. Clone de repository naar uw lokale machine.
2. Zorg ervoor dat u een webserver heeft draaien (zoals Apache), PHP en mysqli zijn ge√Ønstalleerd, en MySQL Server is opgezet.
3. Plaats de bestanden in de root van uw webserver of in een submap.

## Gebruik

Open uw webbrowser en navigeer naar de locatie van de bestanden op uw webserver. U wordt dan naar de loginpagina geleid. Na het inloggen wordt u naar het dashboard geleid waar u uw slimme bijenkorf kunt verbinden.

## Functies

- Inloggen en uitloggen
- Verbinden van een slimme bijenkorf
- Bekijken van gegevens van de slimme bijenkorf
